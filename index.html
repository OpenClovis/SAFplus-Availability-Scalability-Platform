What is it?

The SAFplus Availability Scalability Platform is Service Availability Forum compliant middleware running on Linux and embedded RTOSes. It provides libraries, GUI, ATCA hardware integration, and code generator to design multi-node (clustered) applications that are highly available, redundant, and scalable. Provides node and application fault detection and failover in the 10s of milliseconds, and useful application libraries including distributed hash tables (checkpoint), event, logging, and communications. Implements SA-Forum APIs where applicable. Download stable release with installer from: ftp.openclovis.com

Who Uses it?

Used anywhere reliability and performance is a must -- like telecom, wireless, defense and enterprise cluster computing.
